# Product Backlog

Based on a thorough audit of the application's core directories (specifically `app/Models/`, `app/Http/Controllers/`, and the API/Web route configurations), the existing system architecture and features have been mapped out.

The application is structured around a robust Laravel backend serving a mobile/frontend API (`v1`), alongside a web-based Admin dashboard (using Livewire/Filament). The core modules revolve around Authentication/Onboarding, Activity & Course Scheduling, Reservations, Family Accounts, Subscriptions, Payments (via Konnect), and a Loyalty/Tier system.

Here is the comprehensive Product Backlog reflecting what is currently implemented in the codebase:

| ID | Feature/Module | User Story | Technical Acceptance Criteria | Implementation Status |
| :--- | :--- | :--- | :--- | :--- |
| **US001** | **Auth & Onboarding** | As a user, I want to register and verify my account via OTP so that I can securely access the platform. | - `AuthController@register` handles initial registration.<br>- `AuthController@sendOtp` and `verifyOtp` manage OTP flow.<br>- Models: `User`, `Member`, `OtpCode`. | Fully Implemented |
| **US002** | **Auth & Onboarding** | As a registered user, I want to log in, recover my password, or complete onboarding so that I can manage my account. | - Login, Forgot Password, and Reset Password endpoints exist.<br>- Web route `/member/onboarding-password/{token}` handles onboarding password setup.<br>- Models: `MemberOnboardingToken`. | Fully Implemented |
| **US003** | **Auth & Onboarding** | As an authenticated user, I want to verify my email and phone number to secure my profile. | - Endpoints for `verificationStatus`, `verifyEmail`, and `verifyPhone` exist under `user/`. | Fully Implemented |
| **US004** | **Member Profile** | As a member, I want to view and update my profile information so that my details stay accurate. | - `MemberController@profile` and `updateProfile` handle retrieval and updates.<br>- Endpoint to update password is functional. | Fully Implemented |
